## Comment Format
- There are several different ways to document codes in doxygen format, but Javadoc style, which consists of a C-style comment is recommended to use.

  >```
  >/**
	> * @brief <write here>
	> * @param <write here>
	> * @...
	> * @...
	> * @return <write here>
	>*/
  >```
